# Trabajo Práctico 2 - Módulo 3

Este repositorio contiene algunos ejemplos de uso de módulos de **Node.js** como parte del trabajo práctico del módulo 3
Cada carpeta incluye un pequeño script (`index.mjs`) que demuestra el uso de un módulo del núcleo de Node.js

## Objetivo

El objetivo de este trabajo práctico es familiarizarse con algúnos módulos básicos de Node.js, prácticar la importación y uso de sus funciones.
